Git推送文件至远程仓库(Windows 10)

前提条件：电脑上已经安装Git、拥有一个GitHub帐号

1. 安装Git

git官网

Windows版本的Git：msysgit，下载地址: https://git-for-windows.github.io/
下载下来默认安装即可（也可自行设定安装路径）。

2. 注册GitHub帐号

注册地址：https://github.com/

2.1 设置账户名、账户邮箱及密码，然后点击注册。

2.2 点击继续

2.3 根据需要自行填写相关信息并提交

2.4 点击“开始项目”

2.5 提示验证邮箱，登入邮箱进行验证

3. 创建SSH KEY

在用户主目录下，确认是否有.ssh目录，有的话，再看这个目录下是否有id_rsa和id_rsa.pub这两个文件，有的话，直接跳到下一步。

没有的话，在Cortana中输入Git Bash 点击打开“Git Bash”，创建SSH Key：

然后一直回车（可以设置密码，也可不设置），可以在用户主目录里看到.ssh目录，该目录下有两个文件：id_rsa、id_rsa.pub，分别叫私钥和公钥（公钥可以告诉其他人，私钥不可以）。



4. 将公钥(id_rsa)添加至GitHub

登录GitHub，点击”Settings”,进入后再点击”SSH and GPG keys”, “New SSH key”


在Tilte中填入相关信息，key栏里面填入的是id_rsa.pub（公钥）文件里面的内容。填好后，点击“Add SSH key”即添加完成。
